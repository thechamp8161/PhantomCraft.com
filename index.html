Stripe Checkout Session (Subscription)

This Python/Flask snippet creates a Stripe Checkout session for a new subscription. You can call this endpoint (for example, with different Price IDs for Pro or Gold) to redirect the user to Stripe’s hosted checkout page.


import stripe
from flask import Flask, redirect, request

app = Flask(__name__)

# Set your actual Stripe secret key
stripe.api_key = "sk_test_your_actual_secret_key"  # REPLACE with your Stripe secret key

# For example, use different Price IDs for Pro and Gold subscriptions.
# Replace these with your actual Price IDs from the Stripe Dashboard.
PRO_PRICE_ID = "price_pro_subscription"      # e.g. for €2/month Pro
GOLD_PRICE_ID = "price_gold_subscription"    # e.g. for €3/month Gold

@app.route('/create-checkout-session', methods=['POST'])
def create_checkout_session():
    # Determine the tier from the request (default to Pro)
    tier = request.args.get("tier", "pro").lower()
    if tier == "gold":
        price_id = GOLD_PRICE_ID;
        success_url = "https://yourapp.com/success_gold"  # REPLACE with your Gold success URL
    else:
        price_id = PRO_PRICE_ID;
        success_url = "https://yourapp.com/success"         # REPLACE with your Pro success URL

    try:
        session = stripe.checkout.Session.create(
            payment_method_types=["card"],
            mode="subscription",
            line_items=[{
                "price": price_id,
                "quantity": 1,
            }],
            success_url=success_url,
            cancel_url="https://yourapp.com/cancel",   # REPLACE with your cancel URL
        )
        # Redirect user to the Stripe Checkout session URL
        return redirect(session.url, code=303)
    except Exception as e:
        return str(e)

if __name__ == '__main__':
    app.run(port=3000, debug=True)

Call this endpoint with ?tier=pro for Pro or ?tier=gold for Gold subscriptions.

Modify Subscription: Cancel at Period End

This snippet modifies an active subscription so that it cancels at the end of the current period.


import stripe

stripe.api_key = "sk_test_your_actual_secret_key"  # REPLACE with your secret key

# Replace with the actual Subscription ID you wish to modify:
subscription_id = "sub_your_actual_subscription_id"

# Modify the subscription to cancel at period end
stripe.Subscription.modify(subscription_id, cancel_at_period_end=True)

print("Subscription will cancel at period end.")

Modify Subscription: Update Price

This snippet updates an existing subscription's items with a new Price ID.


import stripe

stripe.api_key = "sk_test_your_actual_secret_key"  # REPLACE with your secret key

# Replace with your actual Subscription ID and new Price ID
subscription_id = "sub_your_actual_subscription_id"
new_price_id = "price_your_new_price_id"  # REPLACE with the new Price ID

# Update the subscription's items with the new Price ID
stripe.Subscription.modify(
    subscription_id,
    items=[{"price": new_price_id}]
)

print("Subscription updated with new price.")
